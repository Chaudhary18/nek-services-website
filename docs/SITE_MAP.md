## Updated Direction

The user wants the next pass to copy the live NEK/Gerow site structure much more closely. The current four-page map remains correct, but the page layouts should be rebuilt around the downloaded reference files in `reference/live-site/`.

Recommended next sitemap:

- Home: Gerow-style finance banner, services intro, about/overview, problems solved, stats, testimonials, client/software logos, CTA.
- About: Gerow-style breadcrumb, founder/company story, mission, specialties, software expertise, team/proof.
- Services: Gerow-style breadcrumb, services grid, detailed service categories, CPA/bookkeeping firm outsourcing section, FAQs, CTA.
- Contact: Gerow-style breadcrumb, contact details, email-only form, consultation copy, LinkedIn link.

Current content already includes the problems solved section, detailed service categories, FAQs, testimonials, logos, and confirmed contact data. The next change should focus on visual similarity to Gerow rather than inventing more content.
